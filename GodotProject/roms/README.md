# ROMs Directory

This folder is for your ZX Spectrum system ROM files.

## Required Files

- **48.rom** (16KB): Standard Sinclair ZX Spectrum 48K ROM.
- **128.rom** (32KB): Combined ROM for the ZX Spectrum 128K model.

## Alternative (Split 128K ROMs)

If you have separate 16KB ROM files for the 128K model, name them:
- **128-0.rom**
- **128-1.rom**

> **Note:** The 48K emulator includes a built-in ROM, but will use `48.rom` from this folder if it is present.

## Where to find ROMs

You can download ZX Spectrum ROMs from:
- [spectrumforeveryone/zx-roms](https://github.com/spectrumforeveryone/zx-roms)
