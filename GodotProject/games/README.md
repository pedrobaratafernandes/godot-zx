# Games Directory

This directory is intended for ZX Spectrum game files.

## How to use

Place your ZX Spectrum game files here. Supported formats are:
- **.tap** (tape)
- **.sna** (snapshot)
- **.z80** (snapshot)

The emulator's **Launcher** will automatically detect all games in this folder and list them in the menu.

## Where to find games

You can find ZX Spectrum games at:
- [esp32rainbow.com](https://www.esp32rainbow.com/games/letter/A?page=1)
